Azure Stream Analytics: 5 new regions are now generally available

Azure Stream Analytics is a fully managed, real-time analytics service designed to help you analyze and process fast moving streams of data that can be used to get insights, build reports, or trigger alerts and actions. The service is now available in 5 new regions.

Azure Stream Analytics is now generally available for customers in 5 new regions:

  • Australia Central
  • Germany West Central
  • Norway East
  • South India
  • UAE North

With this announcement, Stream Analytics is now available in 33 Azure regions worldwide. For more information about local pricing, visit the Azure Stream Analytics pricing webpage.
