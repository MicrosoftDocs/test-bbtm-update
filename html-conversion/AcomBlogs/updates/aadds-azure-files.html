Public preview: Azure AD authentication for Azure Files SMB access

Secure your Azure Files SMB shares by using your Azure AD DS managed domain.

Azure Files offers fully managed file shares in the cloud that are accessible via the industry-standard SMB protocol. Integration with Azure Active Directory Domain Services (Azure AD DS) enables SMB access to Azure file shares using Azure Active Directory (Azure AD) credentials from AD DS domain-joined Windows VMs. In addition, Azure Files supports preserving, inheriting, and enforcing Microsoft file system NTFS ACLs on all folders and files in a file share.

 

This public preview is a collaboration between the Azure Files and Azure Storage teams.
