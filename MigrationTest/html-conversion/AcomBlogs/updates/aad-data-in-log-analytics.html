Send your Azure Active Directory data to Azure Log Analytics

The ability to forward your Azure Active Directory logs to Azure Log Analytics is now available.

One of our top-requested features is available: the ability to forward your Azure Active Directory (Azure AD) logs to Azure Log Analytics. You can now browse, query, visualize, alert on, and do more with your Azure AD log data. For more information, see the documentation.
