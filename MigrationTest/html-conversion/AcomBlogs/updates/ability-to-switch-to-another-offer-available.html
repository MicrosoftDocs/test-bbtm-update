Ability to switch to another offer available

Customers can switch their Pay-As-You-Go subscription to another offer via the Azure Account Center.

Customers can switch their Pay-As-You-Go subscription to another offer via the Azure Account Center. We’ll continue to add more options based on demand. In this rollout, Pay-As-You-Go subscriptions can be switched to one of the following:

  • 12-month commitment, a prepaid offer with a discount on Azure services
  • Any of the monthly Azure credit offers for Visual Studio subscribers
For more information, please visit the Switching to a different Azure offer documentation page.
