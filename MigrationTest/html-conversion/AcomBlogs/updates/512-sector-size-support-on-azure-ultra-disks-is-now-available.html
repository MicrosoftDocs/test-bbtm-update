512 Sector Size support on Azure Ultra Disks is now available

Azure Ultra Disks now support 512 sector sizes.

Azure Ultra Disks offer high throughput, high input/output operations per second (IOPS), and consistent low latency block storage for Azure virtual machines (VMs). Oracle Database earlier than release 12.2 and other workloads that require less than 4K sector size can now benefit from the performance of Ultra Disks.
