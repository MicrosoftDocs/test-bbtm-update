Azure Active Directory B2C—Send customized emails using a custom policy

Use your own third-party email provider to send customized verification emails during sign up or password reset.

Use your own third-party email provider to send customized verification emails during sign up or password reset. This is done by using a custom policy with new elements such as DisplayControls and new REST API technical profile options that allow you to talk to more providers. See how to set this up using SendGrid as an example.
